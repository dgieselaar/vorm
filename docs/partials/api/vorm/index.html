vorm

The vorm module is the core and currently only module for vorm.

Module Components

Directive

Name Description
ngModel

This overloads the ngModel directive, and registers the ngModelController with the vormFieldController and the vormFormController if they're there.

Requires: ngModel, ^?vormField, ^?vormForm
vormChange

Evaluate the given expression when a value changes from the view. It listens to a viewchange event, which is dispatched from a vormFieldController and then bubbles upwards.
vormControlList

This directives manages and displays the available controls.

Requires: ^vormFieldConfig, ^?vormFocusableList, vormField
